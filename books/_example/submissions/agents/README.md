# Agent Submissions Tracker

## Folder Structure

Create one folder per agent, named `lastname-firstname`. Each folder should contain:

```
agents/
  wang-jessica/
    query.md          — Personalized query letter (copied from template, customized)
    notes.md          — Research notes: what they rep, why they fit, MSWL notes
    status.md         — Submission status and dates
    response.md       — Their response (if any) and next steps
  smith-jane/
    ...
```

## Submission Workflow

1. Research the agent — check Publishers Marketplace, QueryTracker, #MSWL, their agency website
2. Create their folder: `agents/lastname-firstname/`
3. Copy the query letter template into `query.md` and personalize
4. Log submission guidelines and research in `notes.md`
5. Track status in `status.md`
6. Query in batches of 8-10. Wait 4-6 weeks between batches.

## Status Codes

- **RESEARCHED** — Agent identified, not yet queried
- **QUERIED** — Query sent, awaiting response
- **PARTIAL** — Agent requested partial manuscript
- **FULL** — Agent requested full manuscript
- **OFFER** — Agent offered representation
- **PASS** — Agent declined
- **NO RESPONSE** — Past the stated response window, assume pass
- **WITHDRAWN** — Query withdrawn (e.g., accepted offer elsewhere)
